### **:bread:前置需求**
+ NodeJS >= 18.16

### :mag:使用方式
```bash
git clone https://github.com/CHENTHIRTEEN/SoftCupA4_Web.git
cd SoftCupA4_Web
pnpm install
pnpm run dev
```
